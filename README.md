# Meta-Geniusz-GOK

## Opis projektu

Repozytorium e-booka "Meta-Geniusz-GOK" to unikalne połączenie publikacji merytorycznej z aktywnym środowiskiem programistycznym. Projekt ten pełni rolę "jednostki centralnej zarządzającej planetą".

### Struktura katalogów
- `/docs` - Pełna treść e-booka podzielona na rozdziały.
- `/src` - Kod źródłowy aplikacji/systemu zarządzającego.
- `/scripts` - Skrypty automatyzacji i wdrożeń.
- `/config` - Pliki konfiguracyjne systemu "Meta-Geniusz".
- `/.github` - Workflowy CI/CD.

## Jak zacząć
1. Sklonuj repozytorium: `git clone https://github.com/sobieranskip95patryk/e-book.git`
2. Przejdź do katalogu projektu: `cd e-book`
3. Zapoznaj się z dokumentacją w folderze `/docs`.
